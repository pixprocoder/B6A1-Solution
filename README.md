What are some differences between interfaces and types in TypeScript?

typescrip এ **টাইপ** and **ইন্টারফেস** আমরা ব্যবহার করি টাইপ ডিক্লেয়ার করার জন্য। সো নিচে কিছু উদাহরণ দেয়া হলো :

| টাইপ                                                   | ইন্টারফেস                            |
| ------------------------------------------------------ | ------------------------------------ |
| এটি রেডিক্লার করা যায় না।                              | এটি রেডিক্লার করা যায় মাল্টিপল টাইম। |
| extend করার জন্য আমাদের **&** ইউস করা লাগে             | এটি তে **extends** ইউস করা লাগে।     |
| এটিতে আমরা primitives, unions, tuples etc.. রাখতে পারি | আমরা Object রাখতে পারি।              |
| এটি একটু স্লোয়ার স্পেসিয়াললি কমপ্লেক্স টাইমস           | একটু ফাস্টার                         |

আমরা কখন ব্যবহার করবো:
**টাইপ**: আমরা মোস্টলী ব্যাবহার করবো unions, primitives, tuples
**ইন্টারফেস**: আমরা মোস্টলী ব্যাবহার করবো Object, classes OOP etc..

আমি কিছু কোড এক্সাম্পল syntex দেয়া হলো  
**টাইপ** :

```ts
type Animal = { name: string };
```

**ইন্টারফেস** :

```ts
interface User {
  name: string;
}
```

উপসংহার : আমরা আমাদের প্রয়োজন অনুসারে ব্যবহার করবো। মোস্টলী অবজেক্ট টিপস অফ things হলে **ইন্টারফেস** use করবো otherwise **টাইপ**
